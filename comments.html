Technical aspects

My site is structured in the traditional single page format which satisfies the simple requirements.

I have also used a number of CSS classes and IDs with their respective selectors to achieve my desired appearance, for example the menu and menud classes are used on each page to apply specific styling to the menu and the home ID selector is used to apply the unique formatting to the home page.
